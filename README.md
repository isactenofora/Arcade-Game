# Arcade game
This game is a classic arcade knowed as 'Frog jump'.
## Getting Started
### Prerequisites
For test the game, you only need a compatible browser and a keyboard for move the character.
### Installing
Installation is not necessary. You only need to click in the `index.html` file.

## Build with
* [Atom](https://atom.io/) - A code editor
* [JavaScript](https://www.javascript.com/) - A language for coding
* [SWAL](https://sweetalert.js.org/) - A beauty library for the alerts messages
